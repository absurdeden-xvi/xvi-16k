# XVI / 十六开

一个为中文写作者设计的文本长图排版器。粘贴文字、调整排版和风格，然后导出可分享的 PNG 长图。

当前阶段：`v0.3` 工作流原型。

## V1 使用方式

直接打开 `index.html` 即可使用，不需要安装依赖或连接服务器。

## 产品原则

- 写作优先：打开后立即可以输入，不用先配置模板。
- 所见即所得：所有调整都即时反映到预览。
- 本地优先：正文与图片均在浏览器中处理。
- 克制但灵活：少量高质量预设，加上关键参数的自由调整。
- 自由优先：首次自动排版提供起点，再次生成保留用户已经调好的版式。
