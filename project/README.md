# XVI / 十六开

一个为中文创作者设计的文本长图排版器。输入完整内容，自动完成基础排版，再调整字体、版面与颜色，最后导出可发布的长图。

当前版本：`v0.6.4`

当前阶段：可用的 V0 工作版本。核心生成流程已经成立，模板体系、视觉细节和长文多图发布仍在继续打磨，尚未进入 V1。

线上版本：[teal-duckanoo-b85abe.netlify.app](https://teal-duckanoo-b85abe.netlify.app/)

## 使用方式

直接访问线上版本即可使用，根网址会直接加载当前正式版本。也可以在本地打开 `index.html`，不需要安装依赖或连接服务器。

1. 输入标题、署名与完整正文。
2. 按需开启文本清理与智能分段。
3. 点击“自动排版”生成长图。
4. 在“样式”中调整字体、字号、间距、颜色与版式细节。
5. 在“保存”中选择 PNG 或 JPG，并按所需清晰度导出。

## 产品原则

- 写作优先：打开后立即可以输入，不用先配置模板。
- 所见即所得：所有调整都即时反映到预览。
- 本地优先：正文与图片均在浏览器中处理。
- 克制但灵活：少量高质量预设，加上关键参数的自由调整。
- 自由优先：首次自动排版提供起点，再次生成保留用户已经调好的版式。

## 开发与发布

项目代码保存在 GitHub 仓库 [`absurdeden-xvi/xvi-16k`](https://github.com/absurdeden-xvi/xvi-16k/tree/main/project)，正式页面由 Netlify 托管。

完成一个阶段的本地修改并确认效果后，建立 Git 版本记录并推送到 GitHub 的 `main` 分支。Netlify 会自动部署新版本，随后再检查线上页面、字体与导出流程是否正常。

当前 Netlify 二级域名可以继续免费使用；独立域名不是上线的必要条件，可在品牌和域名命名确定后再购买并绑定。

版本号统一沿用最初产品线，当前正式版本为 `v0.6.4`。页面、样式和脚本的缓存标记使用同一版本号；`legacy-v0.6.3.html` 仅作为历史版本快照保留。
