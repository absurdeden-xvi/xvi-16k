# XVI / 十六开：产品方向

## 一句话定位

面向中文创作者的智能长图排版工具：粘贴文字，确认偏好，一键得到具有成熟审美且可自由调整的发布图片。

## 品牌原则

现阶段不设置宣传口号。`XVI / 十六开` 本身作为主品牌，避免用空泛文案解释产品。

Freedom 只作为内部产品原则：不需要设计基础也能快速得到可靠结果，同时保留足够的调整空间。

## 核心用户

- 小说、同人、随笔与诗歌作者
- 微博、小红书等平台的内容创作者
- 日记、生活记录和个人表达用户

## V1 不可妥协

1. 自动排版：标题比例、文本清理、分段选项与对齐方式可靠。
2. 模板选择：模板改变配色、字体和完整结构，不只是换背景色。
3. 自由定义：所有关键视觉参数可即时调整并清楚预览。

## 主流程

1. 输入标题、署名与正文。
2. 确认分段、缩进等排版偏好。
3. 一键自动排版。
4. 选择模板或进行即时微调。
5. 导出长图；后续支持按平台自动拆分多图。

## 视觉原则

- 主界面浅色、柔和、低饱和，有情绪但不妨碍工作。
- 使用留白、字体品质和微妙动效建立高级感。
- UI 本身需要有辨识度，因为产品功能聚焦且面向创作者。
- 基础体验简约、清晰；大胆颜色、大字号和非对称结构作为可选模板。
- 避免工业控制台感、刺眼高对比、僵硬方格、冗余装饰和“复古纸张即文学”的陈词滥调。

## 功能层级

### 基础层

- 输入与自动排版
- 高质量默认模板
- 字号、行距、段距、边距与颜色
- PNG / JPG 高清导出

### 进阶层

- 基础与高级设置分层
- 多套推荐方案比较
- 自定义模板保存在浏览器本地
- 平台尺寸预设
- 小红书 3:4 自动拆图与批量保存
- 撤销、重做与自定义快捷键

### 未来探索

- 通过自然语言描述生成风格建议
- AI 配色与版式灵感推荐
- 包浆、像素等特殊视觉效果
- 高级设计能力付费，基础长图转换永久免费

## 暂不进入 V1

- 登录注册与云同步
- 复杂文本语义理解
- 创作社区或内容平台
- PDF、透明字素等低优先级导出

## 安全边界

- 正文默认只在本地浏览器处理。
- 水印、账号和二维码由用户主动添加。
- 发布前明确由用户确认内容与导出结果。
