04 / TYPE CANDIDATES

先确认授权与字集，再谈“好看”。

这一轮只建立候选池，不把字体文件接进正式网页。正文优先完整字集、屏幕阅读和标点质量；标题字体可以更有性格。

建议保留与新增

字体聚合站只负责发现线索。正式内置前，还要回到作者或项目原始页面核对授权、网页嵌入范围与版本。

字体建议用途判断状态
霞鹜新致宋正文 / 标题现有字体中最稳定的现代宋体之一，继续保留；需固定版本并复核网页字体来源。保留
汇文明朝体正文 / 标题气质符合项目，但必须重新核实具体授权与当前 CDN 的再分发条件。待核实
文津宋体长文正文大字符集、OFL，适合生僻字较多的小说与同人正文；视觉个性较弱但可靠。候选
香萃端庄宋体标题 / 短正文重心高、轮廓修长，现代感明显；GB2312 字集限制意味着不宜单独承担所有正文。候选
司源赢宋标题 / 题头窄体与中英混排是优势，适合替代目前老派的标题选项；需测试小字号可读性。候选
宋骨黑体正文 / 现代模板兼具宋体骨架与黑体密度，可补足目前“宋体或普通黑体”之间的空档。候选
文渊黑体UI / 正文多字重与大陆规范字形适合工具界面及现代长文；需要评估网页包体。候选
朱雀仿宋书页 / 信笺继续作为明确风格选项，不作为默认正文，以免所有模板都趋向传统出版物。保留
