# XVI / 十六开

一个为中文创作者设计、在浏览器本地运行的文本长图排版器。完整输入文字后，生成可继续微调并导出的长图。

XVI is a privacy-first, browser-based longform typesetting studio for Chinese writers.

- 在线使用：[xvi-16k.netlify.app](https://xvi-16k.netlify.app/)
- 当前版本：`v0.7.1` 公开预览版
- OpenAI Build Week 赛道：**Apps for Your Life**
- 技术形态：原生 HTML、CSS、JavaScript，无需构建、账户或 API Key
- 隐私边界：正文、导入字体、草稿和图片生成均留在用户浏览器

## 运行

直接打开 [`project/index.html`](./project/index.html)，或访问在线版本。无需安装依赖。

评审可以依次体验：输入中文正文、局部富文本与简繁转换、生成排版、画布内编辑、切换刊页与配色，最后选择格式和清晰度导出。
